# 5LSM0 Assignment
5LSM0 competition on CityScapes dataset

# Getting Started

## Installing
To get started with this project, you need to clone the repository to your local machine. You can do this by running the following command in your terminal:
```
git clone https://github.com/alexandros-kalais/5LSM0-NNCV-Assignments.git
```
After cloning the repository, navigate to the project directory:
``` 
cd 'Final assignment'
```
Install all needed packages
```
pip install -r requirements.txt
```
## File Descriptions:
Here's a brief overview of the files you'll find in this repository:
- losses.py: Includes the loss functions which can be used to evaluate the model
- unet.py: Contains the improved model
- train_combined.py: Conatins the training strategy for the final model
- requirements.txt: Required packages to run the code
- main.sh: Includes the code for tuning the hyperparameters and run the project.
